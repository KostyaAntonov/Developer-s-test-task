Инструкция по запуску

Задание 1: Проверка Email
   1)Установите необходимую библиотеку:
      pip install dnspython
   2)Запустите скрипт из терминала или PyCharm:
      python checker.py

Примечание: Убедитесь, что ваш интернет-провайдер не блокирует порт 25 для корректной работы SMTP-проверки.

Задание 2: Отправка в Telegram

   1)Установите библиотеку для работы с API:
      pip install requests

   2)Откройте файл tg_sender.py и вставьте ваши BOT_TOKEN и CHAT_ID.

   3)Создайте текстовый файл (например, message.txt) с текстом для отправки.

   4)Запустите скрипт, передав путь к файлу:
      python tg_sender.py message.txt

Задание 3:
   Архитектура аутрич-системы (1200 аккаунтов)
   Инфраструктура: 3 VPS-сервера (Ubuntu + Docker). Нагрузка распределяется по 400 аккаунтов на сервер.

   Доменная сетка: ~120 доменов (10 аккаунтов на домен). Обязательная настройка SPF, DKIM и DMARC для минимизации
   спам-фильтров.
   
   Ротация и Прогрев: Лимит 30-50 писем в день с одного ящика. Рандомные задержки (5-15 мин) и предварительный "прогрев"
   аккаунтов (Warm-up) в течение 14 дней.
   
   Мониторинг: Использование Postmaster Tools и мониторинг Bounce Rate. Если Bounce Rate > 3%, аккаунты домена уходят на "
   паузу".
   
   Оценка стоимости: ~ $130-150/мес (основной бюджет — регистрация и продление доменов).
