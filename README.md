## Инструкция по запуску 1 и 2 задания

## Задание 1: Проверка Email
   1. Установите необходимую библиотеку:
      ```bash
      pip install dnspython
   2. Запустите скрипт из терминала или PyCharm:
      ```bash
      python checker.py

- Примечание: Убедитесь, что ваш интернет-провайдер не блокирует порт 25 для корректной работы SMTP-проверки.

## Задание 2: Отправка в Telegram

   1. Установите библиотеку для работы с API:
      ```bash
      pip install dnspython
   2. Откройте файл tg_sender.py и вставьте ваши BOT_TOKEN и CHAT_ID.

   3. Создайте текстовый файл (например, message.txt) с текстом для отправки.

   4. Запустите скрипт, передав путь к файлу:
      python tg_sender.py message.txt

## Задание 3:  Архитектурная задача:

&nbsp;&nbsp;&nbsp;&nbsp;   **Инфраструктура:** 3 VPS-сервера (Ubuntu + Docker). Нагрузка распределяется по 400 аккаунтов на сервер.

&nbsp;&nbsp;&nbsp;&nbsp;   **Доменная сетка:** ~120 доменов (10 аккаунтов на домен). Обязательная настройка SPF, DKIM и DMARC для минимизации
   спам-фильтров.
   
&nbsp;&nbsp;&nbsp;&nbsp;   **Ротация и Прогрев:** Лимит 30-50 писем в день с одного ящика. Рандомные задержки (5-15 мин) и предварительный "прогрев"
   аккаунтов (Warm-up) в течение 14 дней.
   
 &nbsp;&nbsp;&nbsp;&nbsp;  **Мониторинг:** Использование Postmaster Tools и мониторинг Bounce Rate. Если Bounce Rate > 3%, аккаунты домена уходят на "
   паузу".
   
&nbsp;&nbsp;&nbsp;&nbsp;   **Оценка стоимости:** ~ $130-150/мес (основной бюджет — регистрация и продление доменов).

## Задание 4: Блиц: Мой AI-стек: 

&nbsp;&nbsp;&nbsp;&nbsp;   **IDE:** PyCharm Professional (лучшая среда для Python-разработки и дебага).

&nbsp;&nbsp;&nbsp;&nbsp;   **Модели:** Gemini 1.5 Pro через Google AI Studio (для анализа контекста больших файлов) и GPT-4o для быстрого написания
   юнит-тестов и регулярных выражений.
   
&nbsp;&nbsp;&nbsp;&nbsp;   **MCP:** В текущих задачах не использую, предпочитаю прямой контроль над кодом. В основном использую Fetch для быстрого подтягивания свежей документации API.
   
&nbsp;&nbsp;&nbsp;&nbsp;   **Системные правила:** Придерживаюсь PEP8 и обязательной аннотации типов. В общении с AI использую правила: "
   минимум пояснений, максимум чистого кода" и "всегда добавляй обработку таймаутов в сетевые функции".
